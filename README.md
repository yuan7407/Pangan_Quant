# Quantitative Trading Project

本项目为量化交易策略开发与回测平台，包含多版本策略、回测日志、工具脚本等。

## 目录结构
- `src/strategy/` 策略相关代码
- `src/backtest/` 回测相关代码
- `src/utils/` 工具脚本
- `logs/` 回测日志与输出
- `docs/` 项目文档
- `notebooks/` Jupyter分析与可视化

## 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 运行主程序或回测脚本

## 依赖环境
- Python 3.8+
- 见 requirements.txt
